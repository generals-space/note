## 1. 四舍五入求整除

js

```js
let a=18;
a &= ~15; // 16
// 等价于
a = a & ~15 
```

其实应该说是向下取整.

取小于等于`a`的, 可以被16整除的最大值. 比如18不能被16整除, 向下找, 17也不可以, 但是16就可以, 于是得到16.

其中`~15`, 在js中非运算得到`-16`, 可作为除数16. 类似的, `~7`得到`-8`, 可作为除数8.

## 节点插入顺序会影响二叉平衡树的最终形态

两个节点完全相同的树对象, 有可能并不相同. 构造树时节点的插入顺序的不同, 会影响到树的最终形态.

[1, 2, 3, 4]

```
1   =>     2    =>         2          =>           2
         /               /   \                   /   \
        1               1     3                 1     3
                                                        \
                                                         4
```

[4, 3, 2, 1]

```
4   =>     4    =>         3          =>          3
         /               /   \                  /   \
        3               2     4                2     4
                                             /
                                            1
```

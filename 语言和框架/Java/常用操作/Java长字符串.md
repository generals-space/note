# Java长字符串

参考文章

1. [java中有那种长字符串换行的书写方式吗？](https://segmentfault.com/q/1010000020242133)
    - 没有解决.
2. [what if there is multiline string in java ?!](https://blog.zhouzhipeng.com/what-if-there-is-multiline-string-in-java.html)
    - 译名: 如果java也有多行字符串该多好？！

其实就是golang中的反引号

```go
lstr := `
line 1
line 2
...
`
b := 2

```

python中的三引号

```py
lstr = '''
line 1
line 2
...
'''
b = 2
```
